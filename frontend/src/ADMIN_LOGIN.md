## Troubleshooting

**Password Not Working?**
- Make sure you're copying the EXACT password: `Sensual2025Kenya!#Pink`
- Check for extra spaces before/after the password
- Password is case-sensitive (capital S, K, P)
- Try refreshing the page and logging in again

**Admin Panel Not Loading?**
- Check your internet connection
- Clear browser cache and try again
- Open browser console (F12) to check for errors
- Wait 5-10 seconds on first login for account creation

**Logged Out Automatically?**
- Your session expires after a period of inactivity
- Simply log in again with the same password
- Your data is safely stored in the database
